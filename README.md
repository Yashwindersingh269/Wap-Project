# Wap-Project
1. What does this do?
  My Wap project suggests movies, music, quotes, and activities based on user mood.
2. Api's Used
   1. Movie Api
      `https://api.themoviedb.org/3/search/movie?api_key=de77c6d1779b5eb456c35ad08326563e&query=${MOVIE NAME}`
   2. Music Api
      
   3. Quote Api
     
   4. Acitivity Api
     
  
4. Features
  🎭 1. Mood Selection System
    Users select their mood using:
    Emojis 😊😢😴🤯❤️
    Buttons or cards
    Each mood triggers different API data

    🔌 2. Multi-API Data Integration
    🎬 Movies (TMDB)
    🎵 Music (Deezer/Spotify)
    💬 Quotes (Quotable API)
    🎯 Activities (Bored API)
    
    🧾 3. Dynamic Content Cards
    Display results as cards:
    Movie poster + rating
    Song + artist
    Quote text
    Activity suggestion
    
    🔍 4. Search Functionality
    Search movies or songs by keyword
    Instant results
   
    🎯 5. Filtering Options
    Filter by:
    Genre (movies/music)
    Mood category
    Activity type
   
    📊 6. Sorting Feature
    Sort data by:
    Rating ⭐
    Popularity 🔥
    Alphabetical order 🔤
    
    ❤️ 7. Favorites / Like System
    Users can:
    ❤️ Save movies/songs
    View favorites later
    
    🌙 8. Dark Mode / Light Mode
    Toggle theme
    Save preference in localStorage
    
    ⏳ 9. Loading Indicators
    Show spinner/skeleton while fetching API
    
    📱 10. Fully Responsive Design
    Works on:
    Mobile
    Tablet
    Desktop
   
    ⚡ 11. Debounced Search
    Prevents too many API calls while typing
   
    📄 12. Pagination / Infinite Scroll
    Load more results on scroll or button click
   
    🧠 13. Mood History
    Show previously selected moods
   
    🎨 14. Dynamic UI Based on Mood
    Change:
    Background color
    Theme
    Animations
   
    🔁 15. Refresh Recommendations Button
    Get new suggestions instantly
   
    📌 16. “View More Details”
    Expand card for:
    Movie overview
    Song preview
    Full quote
   
    🧩 17. Category Tabs
    Switch between:
    Movies | Music | Quotes | Activities
